Fastjson初探

记录一些Fastjson的使用方法、特征、原理

Fastjson 简介

Fastjson 是阿里巴巴的开源 JSON 解析库，可以将 Java 对象转换为 JSON 格式，当然它也可以将 JSON 字符串转换为 Java 对象。

Fastjson 可以操作任何 Java 对象，即使是一些预先存在的没有源码的对象。

public的属性会进行反序列化赋值，private的属性不会直接进行反序列化复制，而是调用setxxx（xxx为属性名）的函数进行赋值
